모바일 주요 기능

Firefox의 모든 장점을 모바일 기기에서 만나보실 수 있습니다.

빠름

빠른 실행 속도와 페이지 로딩이 웹 서핑을 쾌적하게 만들어 줍니다.

The 스마트 주소창이 웹 사이트 이용 내역을 기억해 두고 있어서 검색이 보다 쉬워지고 원하는 곳을 빠르게 이동할 수 있습니다.

안전함

개인 정보 추적 금지와 같은 기능이 웹 사이트가 추적할 수 있는 개인 정보 범위를 조절합니다. 자세히 알아보기

최신 버전을 사용하시면 보안 설정에서 비밀번호, 개인 정보, 쿠키, 이미지, 부가 기능의 보안 설정을 간편하게 조정할 수 있습니다.

똑똑함

HTML5와 웹 API 완벽 지원을 통해 모바일 웹의 끝없는 가능성을 경험해 보십시오. 자세히 알아보기

데스크톱 버전에 저장된 이용 내역과 비밀번호를 모바일 장치와 동기화하면 모바일에서도 익숙한 웹 서핑 경험을 할 수 있습니다.

탭에 사이트 화면이 썸네일 이미지로 표시되서 탭 전환이 매우 편리합니다.
